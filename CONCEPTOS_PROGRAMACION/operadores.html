Operadores

Tipos de operadores

Se pueden encontramos tres tipos:

Operadores aritméticos

Exponenciación: ^

Producto y división: * y /

Suma y resta: + y -

Operadores relacionales o de comparación

Se llaman operadores relacionales u operadores de comparación a aquellos que permiten comparar dos valores evaluando si se relacionan cumpliendo el ser menor uno que otro, mayor uno que otro, igual uno que otro, etc.

Mayor: >
Mayor o igual: >=
Menor: <
Menor o igual: <=
Igual: ==
Distinto: <>

Operadores lógicos

Los operadores lógicos permiten además introducir nexos entre condiciones como “y se cumple también que” ó

Y: AND (&)
O: OR ( ||)
NO: NOT ( !=, <>, etc.)
O exclusivo: XOR

Tarea

1. Dadas las variables de tipo entero con valores A = 5, B = 3, C = -12 indicar si la evaluación de estas expresiones daría como resultado verdadero o falso:

a) A > 3                                                            b) A > C                                               c) A < C

d) B < C                                                            e) B != C                                               f) A == 3

g) A * B == 15                                                   h) A * B == -30                                    i) C / B < A

j) C / B == -10                                                    k) C / B == -4                                       l) A + B + C == 5

m) (A+B == 8) && (A-B == 2)                       n) (A+B == 8) || (A-B == 6)

o) A > 3 && B > 3 && C < 3                           p) A > 3 && B >= 3 && C < -3
